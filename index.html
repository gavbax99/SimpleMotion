SimpleMotion

A simple  JavaScipt library  to animate  HTML elements  on-scroll. 

View on Github

Scroll down!

Give Your Elements Motion

Simple to implement and easy to use. Link the JavaScript file above the closing body tag of your HTML and add a data-sm attribute to any element you want to animate.

By default, SimpleMotion-enabled elements fade-in as the user scrolls them into view (like the colored bars on the right). However, you can customize:

  • Distance travelled

  • Animation duration

  • Ending CSS transform

  • Delay before animating

  • Starting and ending opacity

SimpleMotion Examples:

1

Transform distance

                
<p
  data-sm
  data-sm-dist-x="50"
  data-sm-dist-y="0"
  class="example-dist">
  Moves left!
</p>

<p
  data-sm
  data-sm-dist-x="-50"
  data-sm-dist-y="0"
  class="example-dist">
  Moves right!
</p>

4

Custom ending transform

                
<p
  data-sm
  data-sm-dist-x="-200"
  data-sm-dist-y="100"
  data-sm-time="2s"
  data-sm-final-transform=
    "translate(0, 0)
    scale(1.5)
    rotate(360deg)"
  class="example-transform">
  Custom transform!
</p>
